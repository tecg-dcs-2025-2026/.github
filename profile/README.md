# Développement côté serveur - édition 2025-2026

Cette organisation sert de dépôt de code pour le cours de développement côté serveur donné en B2, orientation web, dans le bachelier en techniques graphiques de la [Haute École de la province de Liège](https://hepl.be)
Le cours en question propose une initiation à la programmation d'un serveur web à l'aide du langage PHP.

Quelques resources utiles : 

- [Herd](https://herd.laravel.com) permet de disposer de tout le nécessaire en une seule installation ;
- [DBNgin](https://dbngin.com) permet de disposer d'un installateur de SGBD pratique. Dans le cadre de ce cours, nous n'en aurons sans doute pas besoin. SQLite suffira ;
- [La documentation de PHP](https://php.net) est une des meilleures documentation disponible sur le marché des langages ;
- [Laracasts](https://laracasts.com) est une plateforme de screencast d'excellente qualité autour de l'écosystème du framework Laravel et contient des séries pour les débutants, notamment en PHP ;
- [Laravel](https://laravel.com) est un framework de référence pour le développement côté serveur. Construit sur PHP, il nous fournira des guides dans la découverte des points importants pour notre apprentissage. 

Bon amusement !
